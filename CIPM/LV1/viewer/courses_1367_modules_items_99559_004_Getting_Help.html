Module 0: Getting Started 0.04 | Getting Help

The dedicated Have a question? Ask the Community! discussion forum enables the learning community to benefit from seeing the question asked and the answer provided. Others will likely have the same question, so you are encouraged to use that forum. If you know the answer to a question being asked, you are encouraged to respond.

Use the following guidelines to help you determine how to get the help you need:

  • Find answers to Frequently Asked Questions about your account, program registration, the CIPM exam, and CFA Institute membership on our support page.
  • Have a general question about the Program? Use the Have a Question? Ask the Community! discussion forum. The CIPM Program Staff team will monitor this forum. Please note that CIPM Program Staff cannot answer content-related questions.  
  • Have any other non-content-related questions? Message the CIPM Program Staff directly via the Inbox.

For technical issues with the learning platform, please reach out to Canvas support via the ‘Help’ icon () in the global navigation menu on the left side of this window. It is available 24/7, 365 days per year. We recommend using the chat function for the quickest support.
